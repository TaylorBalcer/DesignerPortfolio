Promotional Site - British Columbia

Talent Showcased: Design and Front-end Development

Objective: Promote British Columbia

Target Audience: Adults

HIGHLIGHTS:

  • An abundance of carefully chosen images, showcase the indigenious culture represented throughout British Columbia, as well as attractive sites like the cities and ocean-views
  • CSS Grids were used for the content structure, so that it can rearrange responsivley to fit any screen size
  • Created my own videos for the project. Then functional audio and video elements and controls were coded onto the page to make users feel immersed in the locale. They can listen to and see British Columbia, from Vancouver all the way to the Sunshine Coast in Pender Harbour.

WordPress - Foraging Blog

Talent Showcased: Design

Objective: Inform about the uses and practices of foraging and food sustainability.

Target Audience: Teenagers to adults

HIGHLIGHTS:

  • WordPress was used for this project to start a blog with e commerce capabilites which came with a premade template that was later changed by creating a custom theme.
  • Imagery of plants, mushrooms and herbs and a color scheme or purple, greens, and golds were carefully selected to set the bright & lively mood for the blog.
  • Decorative motifs involving mushrooms and floral shapes are repeated as visual style elements to provide consistency with the topic of the website.
  • A custom theme was carefully developed to let the user easily see all information on mupltiple screens, purchase products, and interact with other users.
