Promotional Site - British Columbia

TALENT SHOWCASED

Web Design and Front-End Development

OBJECTIVE

Promote British Columbia

TARGET AUDIENCE

Adults

HIGHLIGHTS

  • An abundance of carefully chosen images showcase the indigenous cultures represented throughout British Columbia, as well as attractive sites like the cities and oceanviews.
  • CSS Grids were used for the content structure so that it can rearrange responsively to fit any screen size.
  • I created my own videos for the project. Then functional audio and video elements and controls were coded onto the page to make users feel immersed in the locale. They can listen to and see British Columbia, from Vancouver all the way to the Sunshine Coast in Pender Harbour.

Interactive - Goldenne Fruit Smoothie Builder

TALENT SHOWCASED

Design and Front/Back-End Development

OBJECTIVE

Allow user to make a custom order to pick up from a Smoothie shop

TARGET AUDIENCE

Teenagers, adults

HIGHLIGHTS

  • A bright and beautiful color palette was used to create a fresh and positive mood and was inspired by the colors of fruit.
  • The app retains the user's name and guides them through several screens of choices to build and customize a product to their liking. With each selection, the user can see the cart update, and hand-coded JavaScript places the information in local storage, so it can be displayed during checkout.
  • The User Interface was carefully designed with CSS to fit major elements on one screen for every screen size and orientation.

WordPress - Feral Foraging Blog

TALENT SHOWCASED

WordPress Development & Design

OBJECTIVE

Inform about the uses and practices of foraging and food sustainability.

TARGET AUDIENCE

Teenagers, adults

HIGHLIGHTS

  • WordPress was used for this project to give the site owner the ability to easily create and maintain a blog, and plugins were added to give him an E-Commerce component for selling merchandise.
  • A color scheme of purple, pink and beige was carefully selected to set a bright and lively mood for the blog.
  • A custom theme was carefully developed to permit full control over layout and style customization, including the development of a designed header, sidebar, and navigation menu.
  • Plugins were added to allow enhanced content and functionality, such as WooCommerce for an online shop, MetaSlider for an informational slideshow, and User Registration to let users create an account and post comments to interact with other users.
